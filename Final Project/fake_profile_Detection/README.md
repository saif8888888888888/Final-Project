# Fake Profile Detection System

A comprehensive web application for detecting fake social media profiles using rule-based analysis and machine learning techniques.

## Features

- **Real-time Profile Analysis**: Analyze social media profiles across multiple platforms
- **Rule-based Detection Engine**: Uses multiple heuristics to identify fake profiles
- **User Dashboard**: Track scan history and analytics
- **Secure Authentication**: User registration and login system
- **Responsive Design**: Works on desktop and mobile devices

## Supported Platforms

- Instagram
- Facebook  
- Twitter
- TikTok
